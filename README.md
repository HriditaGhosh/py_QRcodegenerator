# py-_QRcodegenerator
A feature-rich Python QR Code Generator with customizable colors, shapes, and scanning capabilities using Tkinter and qrcode libraries. Supports saving, scanning, and auto-opening URLs.
📝 Python QR Code Generator  
A simple Python Tkinter-based GUI application to generate and scan QR codes with customizable colors, shapes, and styles. Also supports saving QR codes and auto-opening URLs.

🚀 Features
- Generate QR codes from text, URLs, or data
- Customizable colors, shapes, and styles
- Save QR code as PNG image
- Scan QR codes using your webcam
- Auto-open URLs from scanned QR codes
- Optional: Executable version (.exe) – Run the app without installing Python

🛠️ Built With
- Language: Python  
- GUI Framework: Tkinter  
- QR Code Library: qrcode, Pillow  
- QR Code Scanner: pyzbar, OpenCV  
- Build Tool: PyInstaller (for .exe)  
- Version Control: Git & GitHub

▶️ How to Run
1. Install Python  
2. Run:python pythonQRcodegenerator.py
